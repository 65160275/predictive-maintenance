# Predictive Maintenance Analysis

This project analyzes vibration data of rotating machinery based on ISO 10816-3.

Machines analyzed:
- Jockey Pump
- Cooling Pump
- Compressor Motor

Method:
1. Extract vibration amplitude
2. Calculate peak vibration
3. Analyze vibration trend
4. Compare machine condition

Tools:
- Python
- Matplotlib

Output:
- Vibration waveform graphs
- Trend analysis
- Machine comparison
